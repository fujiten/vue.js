算出プロパティとメソッドの違い

  • computedは()不要、methodsは必要
  • computedはgetter,setter定義可能、methodsはgetterのみ
  • computedはキャッシュされる。methodsはされない
算出プロパティのsetter,getterの記述法 computed: { taxIncludedPrice: { get: function(){ return parseInt(this.basePrice * 1.08) }, set: function(taxIncludedPrice){ this.basePrice = Math.ceil(taxIncludedPrice / 1.08) } } } methods: { returnRandomNumber: function(){ return Math.random() } }, computed: { returnRandomNumber: function(){ return Math.random() } }
